# 学习路径

本文件面向第一次进入仓库的学习者，给出几条可执行的阅读路线作为建议。笔记内容覆盖面很广，不建议逐篇全部阅读。

## 路线 A：人工智能入门

适合已有基础数学和编程能力，但还没有系统学习深度学习的读者。

1. 第1部分：1.深度学习基础理论
2. 第1部分：2.神经网络训练的常用方法
3. 第1部分：3.1 卷积神经网络（CNN）
4. 第1部分：3.4 残差网络（ResNet）
5. 第3部分：15.注意力机制与Transformer
6. 第3部分：16.大语言模型的基本原理
7. 第4部分：22.大模型智能体

## 路线 B：强化学习

适合希望理解经典 RL、RLHF、PPO、GRPO 等内容的读者。

1. 第2部分：10.强化学习的基本知识
2. 第2部分：11.基于价值的强化学习
3. 第2部分：12.基于策略的强化学习
4. 第2部分：13.综合价值与策略的算法
5. 第3部分：17.强化微调

## 路线 C：大语言模型与智能体

适合希望系统理解现代 LLM 架构、训练、推理和 Agent 的读者。

1. 第3部分：15.注意力机制与Transformer
2. 第3部分：16.大语言模型的基本原理
3. 第3部分：17.强化微调
4. 第3部分：19.注意力机制的工程优化
5. 第4部分：22.大模型智能体
6. 第4部分：23.工具调用
7. 第4部分：24.大模型推理
8. 第4部分：25.检索增强生成
9. 第4部分：26.上下文与记忆

## 路线 D：世界模型、多模态生成与具身智能

适合已经具备深度学习、Transformer 和生成模型基础，想了解世界模型、多模态与具身智能等前沿研究方向的读者。

1. 第5部分：30.扩散模型与流匹配模型的原理和架构
2. 第5部分：32.世界模型的基本知识
3. 第5部分：33.具身智能的基本知识
4. 第5部分：34-39 前沿研究的各方向或技术路线，可根据自身兴趣选读

## 阅读提示

- 标题含“论文”的文档通常是前沿论文阅读笔记，不代表领域共识或业界标准做法。
